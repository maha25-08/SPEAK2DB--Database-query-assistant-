Query Help
Allowed Query Types
  • SELECT - Retrieve data from database tables
  • SELECT with WHERE - Filter results with conditions
  • SELECT with JOIN - Combine data from multiple tables
  • SELECT with ORDER BY - Sort results
  • SELECT with LIMIT - Limit number of results
Security Restrictions
For security reasons, only SELECT queries are allowed.
Example Queries
Find Books by Title
SELECT * FROM Books WHERE title LIKE '%search%' ORDER BY title;
Get Issued Books
SELECT i.*, b.title FROM Issued i JOIN Books b ON i.book_id = b.id WHERE i.return_date IS NULL;
Count Books by Category
SELECT category, COUNT(*) as count FROM Books GROUP BY category ORDER BY count DESC;
Find Overdue Books
SELECT i.*, b.title FROM Issued i JOIN Books b ON i.book_id = b.id WHERE i.return_date IS NULL AND i.due_date < date('now');
